MRUE

Most
Recently
Used
Enumerator

Description:
This tool is a modified version of RegRipper that will locate 
and enumerate common MRU lists that have been created by both 
Windows and other applications; it will then take all results 
and visualize them using D3.js for easier consumption.

Prerequisites:
	Perl
	Web browser
	Windows registry files
	D3.min.js library (included)
